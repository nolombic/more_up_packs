# more_up_packs 
Upgrade your upgrade packs so you can have even higher stats!  
Affected stats are health and breath.  

## Higher stats, whaaaa?
[The original upgrade_packs mod by Krock](https://content.minetest.net/packages/Krock/upgrade_packs/ "Link to the upgrade_packs minetest ContentDB site") added four slots to the players inventory which they were able to fill with either +10 Health or +5 Breath each.  
Using only that mod a player could have up to +40HP or +20 Breath.  
I however thought that this wouldn't be enough for **some** reason.  

### So I've added more packs.
Introducing:  
>#### ![green heart](https://mup-dev.pltrq.at/custom/green_heart.png) +20HP  
>#### ![blue heart](https://mup-dev.pltrq.at/custom/blue_heart.png) +30HP  
>#### ![violet heart](https://mup-dev.pltrq.at/custom/violet_heart.png) +40HP  
>#### ![golden heart](https://mup-dev.pltrq.at/custom/golden_heart.png) +50HP  
>as well as  
>#### ![violet bubble](https://mup-dev.pltrq.at/custom/violet_bubble.png) +10 Breath  
>#### ![pink bubble](https://mup-dev.pltrq.at/custom/pink_bubble.png) +15 Breath  
>#### ![golden bubble](https://mup-dev.pltrq.at/custom/golden_bubble.png) +20 Breath

## That's all fine and dandy but how do you even craft them?
In order to craft one of these packs you take the previous level and surround it with the right items in order to upgrade the upgrade pack.

### ![red heart](https://mup-dev.pltrq.at/default/heart.png) +10HP (from [the original upgrade_packs mod by Krock](https://content.minetest.net/packages/Krock/upgrade_packs/ "Link to the upgrade_packs minetest ContentDB site")) 

>To craft a ![red heart](https://mup-dev.pltrq.at/default/heart.png) +10HP upgrade pack, you'll need:  
>4 ![copper ingot](https://mup-dev.pltrq.at/default/copper_ingot.png) copper ingots  
>4 ![mese crystal](https://mup-dev.pltrq.at/default/mese_crystal.png) mese crystals  
>1 ![glass bottle](https://mup-dev.pltrq.at/default/glass_bottle.png) glass bottle  
>
>#### Arrange it like this:
>
>>![copper ingot](https://mup-dev.pltrq.at/default/copper_ingot.png) ![mese crystal](https://mup-dev.pltrq.at/default/mese_crystal.png) ![copper ingot](https://mup-dev.pltrq.at/default/copper_ingot.png)  
>>![mese crystal](https://mup-dev.pltrq.at/default/mese_crystal.png) ![glass bottle](https://mup-dev.pltrq.at/default/glass_bottle.png) ![mese crystal](https://mup-dev.pltrq.at/default/mese_crystal.png)  
>>![copper ingot](https://mup-dev.pltrq.at/default/copper_ingot.png) ![mese crystal](https://mup-dev.pltrq.at/default/mese_crystal.png) ![copper ingot](https://mup-dev.pltrq.at/default/copper_ingot.png)  
>>  
>>Put copper ingots into the four corners of the crafting grid. Put an empty glass bottle into the middle spot. Fill in the rest with mese crystals.  
>
>#### Unlocked award IF [awards](https://content.minetest.net/packages/rubenwardy/awards/ "Link to the awards minetest ContentDB site") is used:
>
>>![award for red heart](https://mup-dev.pltrq.at/custom/award_hp10.png)  
>>**Rejuvination**

### Translation template (in case you want to help)

File name: `more_up_packs.[??].tr`  
```
# textdomain:more_up_packs  
HP=  
Breath=  
Craft a @1 @2 Upgrade Pack=
Rejuvination=  
Prolonged Life=  
Fountain of Life=  
Approaching Invincibility=  
Cheating Death=  
Breating in Deeply=  
Becoming Amphibious=  
Growing Gills=  
Underwater Breating=  
More Upgrade Packs=  
Upgrade your upgrade packs so you can have even higher stats!=  
```  
Fill in `[??]` with your ISO language code  
For example: `more_up_packs.nl.tr` or `more_up_packs.de.tr`  
`@1` represents a numeric value like +20  
`@2` represents the pack's type (Health/HP, Breath)

## Last but not least: Installation and dependencies

Installation of a mod must not be hard. Simply head into your minetest 5.0+ game and go to the ***Content*** tab. Tap on ***browse online content*** and wait for it to load up the available mod list. Using the search bar you have to make sure to install these mods:  
  
### [upgrade_packs](https://content.minetest.net/packages/Krock/upgrade_packs/ "Link to the upgrade_packs minetest ContentDB site")
the base that brings the core functionality. Must-have for this mod.  
### [mtg_plus](https://content.minetest.net/packages/Wuzzy/mtg_plus/ "Link to the mtg_plus minetest ContentDB site")
optionally used for the +50HP upgrade pack to make it harder to craft.  
### [awards](https://content.minetest.net/packages/rubenwardy/awards/ "Link to the awards minetest ContentDB site")
if you'd like to be rewarded by a popup for your efforts.
### [more_up_packs](https://content.minetest.net/packages/nolombic/more_up_packs/)
this mod.  

### Make sure the mods are enabled on your world! Otherwise they won't do anything but sit around unused.

Alternatively download the individual mods from their respective pages, extract their .zip files, maybe rename them accordingly, and move them to your mods folder.
